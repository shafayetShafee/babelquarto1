# Hosting a Multilingual Quarto Book in Github Pages (using the  `{babelquarto}` R package)

Suppose you have a book directory with the quarto docs in different languages. The directory sturcture is as follows,

``` bash
blop
├── cover.png
├── index.es.qmd
├── index.fr.qmd
├── index.qmd
├── intro.es.qmd
├── intro.fr.qmd
├── intro.qmd
├── references.bib
├── references.es.qmd
├── references.fr.qmd
├── references.qmd
├── summary.es.qmd
├── summary.fr.qmd
├── summary.qmd
└── _quarto.yml
```
